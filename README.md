# High Performance Python Labs

## Быстрый старт

1. Перейдите в папку третьей лабораторной работы:
   ```bash
   cd labs/lab03
   ```
2. Запустите сборку и старт контейнеров:
   ```bash
   docker-compose up --build
   ```
3. Откройте в браузере ссылку:
   **http://localhost:8000**

## Структура проекта и запуск вручную

Если вы хотите запускать лабораторные работы по отдельности локально, следуйте инструкциям ниже.

### Предварительные требования
*   Python 3.12+
*   Poetry
*   Компилятор C++ (MSVC для Windows или g++ для Linux)

### Лабораторная работа №1: Python vs NumPy
Сравнение стандартных списков и векторизации NumPy.

*   **Расположение:** `labs/lab01/`
*   **Отчет:** `labs/lab01/Отчет.md`
*   **Запуск:**
    ```bash
    cd labs/lab01
    poetry install
    poetry run python lab01.py
    ```

### Лабораторная работа №2: Numba vs PyBind11 (C++)
Сравнение JIT-компиляции и нативного C++ расширения.

*   **Расположение:** `labs/lab02/`
*   **Отчет:** `labs/lab02/Отчет.md`
*   **Компиляция и запуск:**
    В этой работе используется C++ расширение, которое необходимо скомпилировать перед запуском.
    
    1. Установите зависимости:
       ```bash
       cd labs/lab02
       poetry install --no-root --no-interaction --no-ansi --no-root
       ```
    2. Скомпилируйте модуль (создастся .pyd или .so файл):
       ```bash
       poetry run python build.py build_ext --inplace
       ```
    3. Запустите основной скрипт:
       ```bash
       poetry run python lab02.py
       ```

### Лабораторная работа №3: Docker & Orchestration
Упаковка решений в контейнеры и вывод результатов через веб-сервер.

*   **Расположение:** `labs/lab03/`
*   **Отчет:** `labs/lab03/Отчет.md`
*   **Запуск:** См. раздел "Быстрый старт".
